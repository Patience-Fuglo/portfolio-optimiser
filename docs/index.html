4. Hierarchical Risk Parity New in v3.0

Why HRP?

Classical MVO amplifies covariance estimation errors through matrix inversion. HRP sidesteps inversion entirely by using the structure of the covariance matrix — via hierarchical clustering — to guide capital allocation.

Algorithm

StepOperationFormula / Tool
1. DistanceCorrelation → distance matrixd_ij = √(0.5·(1−ρ_ij))
2. ClusterSingle-linkage hierarchical clusteringscipy.cluster.hierarchy.linkage
3. Quasi-diag.Sort assets by cluster leaf orderleaves_list(link)
4. BisectRecursive bisection — allocate inversely to cluster varianceα = V_right / (V_left + V_right)

Dendrogram

Single-linkage clustering groups assets by correlation distance. Capital is then recursively bisected across branches — tightly correlated clusters receive combined allocations before internal splitting.

HRP consistently outperforms MVO out-of-sample on large universes where covariance estimation error dominates. López de Prado's 2016 paper is one of the most cited in modern quantitative finance.
Industry use: AQR, Man Group, and major systematic funds use hierarchical clustering methods for portfolio construction and risk management.

5. Black-Litterman Model

Blends CAPM equilibrium returns with investor views using Bayes' theorem.

π = δ × Σ × w_market (CAPM-implied equilibrium returns)
Ω = τ × P × Σ × Pᵀ (Meucci proportional view uncertainty)
μ_BL = [(τΣ)⁻¹ + PᵀΩ⁻¹P]⁻¹ [(τΣ)⁻¹π + PᵀΩ⁻¹Q]
# Absolute: AAPL returns 25% p.a.   |   Relative: MSFT outperforms GOOGL by 5%
P = np.array([[1, 0, 0, 0, 0], [0, 1,-1, 0, 0]])
Q = np.array([0.25, 0.05])
mu_bl = black_litterman(cov, market_weights, P, Q)
weights = max_sharpe_ratio(mu_bl, cov)     # naturally diversified result
Black-Litterman produces diversified allocations even without constraints. Used at Goldman Sachs, BlackRock, and PIMCO as the standard return-forecasting framework.

6. Factor Risk Model (FF3) New in v3.0

Model

r_i = α_i + β_i·Mkt-RF + s_i·SMB + h_i·HML + ε_i
Σ_f = B × F × Bᵀ + D (factor cov + diagonal idiosyncratic)

B = loadings (n×k) via OLS  ·  F = annualised factor cov  ·  D = diagonal idiosyncratic variance

8. Risk Attribution & CVaR

RC_i = w_i × (Σw)_i / σ_p (Euler decomposition — sums to σ_p)
CVaR_α = −E[R | R ≤ VaR_α] annualised: CVaR_daily × √252
CVaR (43.92%) is 2× the annualised volatility (20.51%) — fat-tailed tech returns mean Gaussian assumptions severely understate tail risk.
